# 🧪 Farsi Periodic Table (جدول تناوبی فارسی)

Welcome! This repository hosts a clean, responsive, and interactive Periodic Table of Elements fully translated and optimized for Persian (Farsi) speakers. 

Whether you are a student, educator, or developer, this project is designed to be a lightweight, open-source resource for learning chemistry in Persian.

## 🚀 Features
- **Fully Localized:** All element names, symbols, and details in Farsi.
- **Responsive Design:** Works flawlessly on mobile, tablet, and desktop screens.
- **Built with Astro:** Highly optimized, fast loading, and static-page ready.

## 🤝 Contribution (On Your Level!)
This is an open public repository, and contributions of all skill levels are welcome! You don't need to be a coding expert to help out. 

Here is how you can contribute:
- ✍️ **Fixing Typos:** Notice a translation error or misspelled chemical term? Open an issue or PR!
- 🎨 **Styling & UI:** Help make the table look even better by tweaking the CSS or adding new themes.
- 🧪 **Data Accuracy:** Double-check atomic weights, categories, or discovery years.

To get started locally:
1. Fork the repository
2. Run `npm install`
3. Run `npm run dev` to start the local Astro server
